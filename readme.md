**项目简介**
本仓库致力于为Half-Life 1经典模组“Field Intensity”提供简体中文本地化支持，包括但不限于字幕、对话和其他相关文本内容的翻译工作。模组原项目可在ModDB上找到：[Field Intensity ModDB页面](https://www.moddb.com/mods/field-intensity)
